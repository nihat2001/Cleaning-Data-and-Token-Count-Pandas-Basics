**Text Preprocessing & Chunking Pipeline.**

**This project is a Python-based application that demonstrates fundamental text preprocessing and data chunking techniques for NLP tasks. It processes raw textual data using regular expressions (Regex) to remove HTML tags, URLs, and redundant whitespaces, preparing it for downstream language model applications.**

**By leveraging the `pandas` library for structured data management and OpenAI's `tiktoken` for precise token count analysis, the script efficiently breaks down long texts into overlapping segments (chunks). It serves as a practical example of building a foundational data pipeline for RAG (Retrieval-Augmented Generation) workflows, highlighting metadata tracking such as source tracking, character lengths, and token counts.**
